16. What will be the width of the div element in the given code snippet ?

div {
  width: 320px;
  padding: 10px;
  border: 5px solid blue;
  margin: 0;
}

A. 360 px
B. 350 px
C. 310 px
D. None of the above


ANSWER: B
Explanation: 320 px (width) + 10 * 2 px ( right and left padding ) + 5 * 2 px (left + right border) + 0px (margin) = 350px
