#CSSBoxModel

1. Which CSS property is used to specify the maximum width of an elements content box ?

A. max-height property
B. max-width property
C. width property
D. height property

ANSWER: B
Explanation: max-width property in CSS is used to specify the maximum width of an element.


2. The CSS property is used to set the maximum height of the element's content box.

A. height property
B. max-height property
C. max-width property
D. None of the Above

ANSWER: B 
Explanation: max-height property is use to set the maximum height of the element's content box.


3. Which of the CSS property is used to set the minimum width of the element's content box ?

A. width property
B. min-width property
C. max-height property
D. None of the above

ANSWER: B 
Explanation: min-width property is used to set the minimum width of an element's content box.


4. How to set the horizontal alignment of a table-cell box or the block element in CSS ?

A. text-transform
B. text-align
C. text-decoration
D. text-shadow

ANSWER: B
Explanation: text-align is used to define the horizontal alignment of a table-cell box or the block element.


5. How to define the order of the flex item in the grid container ?

A. float 
B. order
C. overflow
D. None of the above

ANSWER: B 
Explanation: order proeprty is used to define the order of the flex item in the grid container and it ordering the flex items.


6. What are the components of the CSS box model ?

A. borders
B. margins
C. padding
D. All the above

ANSWER: D
Explanation: Borders, Margins and Padding are the parts of the CSS box model.
