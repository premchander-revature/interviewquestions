1.How does the picture attribute work?

A.Set an image as submit button
B.Bring default image to the page
C.Sets an image background
D.Set an image anywhere on the page

Answer: A

Explanation: To set an image as the submit button using the image input type and  The syntax is : <input type="image" src="imghj.gif" alt="Submit">.  This makes the picture supplied by url the submit button. By pressing the submit button, the image is triggered by sending the X and Y coordinates of the click.
