1.The input ____________attribute specifies the visible width, in characters, of an input field.

A.Disabled
B.Readonly 
C.Size
D.Value

Answer: C

Explanation: The size attribute specifies the width and height of the both <input> and <select> elements. If the type attribute is text or password, then the amount of characters is used for the input.

2.The input _____ attribute specifies the maximum number of characters allowed in an input field.

A.size
B.maxlength
C.min and max
D.multiple

Answer: B

Explanation: The max length attribute specifies the number of characters as (in UTF-16 code units) that a user can submit into an <input> or <text area>. This must be an integer value of zero or greater than that. The input or text area has no maximum length if no max length is given or an erroneous value is specified.

3.The input _________ attributes specify the minimum and maximum values for an input field.

A.disabled
B.maxlength
C.min and max
D.size 

Answer: C

Explanation: The min and max attributes are used to define the <input> element's value range. The min attribute specifies the least value for the< input> element, while the max attribute specifies the maximum value for the <input> element.
